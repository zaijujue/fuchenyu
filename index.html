/ 成长历程

从第一个起降，到第一百次起飞

没有一条路是捷径。但每一条走过的路，都让我变得更强。

2022

入读河南地矿职业学院，加入22级无人机一班

第一次握遥控器的时候，手在抖。三个月后，我能在30秒内完成全套起降流程。不是因为天赋——是因为别人放假出去玩的时候，我还在飞场上练到天黑。

2023

无人机技术全面爆发

这一年，我掌握了航测规划、正射影像、三维点云重建全套流程。从单兵作业到带队测绘，从平原到山地，精度从未掉过线。别人飞的是兴趣，我飞的是专业。

2024

跨界编程——从零到全栈

一架无人机可以飞多高？我决定试试别的答案。自学 Python、HTML、CSS、JavaScript、Node.js。三个月后，第一个 Web 应用上线。六个月后，我已经能独立交付全栈项目。不是聪明——是每天写到凌晨三点。

记事日历 PWA 上线 GitHub Pages，从开发到部署一气呵成。AI Agent 工作流覆盖系统运维全场景。这一年我证明了：不需要团队、不需要背景、不需要资源——只要够拼，一个人能完成五个人做的事。

技术栈持续扩展，产品矩阵不断丰富。从无人机到代码，从本地到云端，从单兵到全能。今年要做的不是提高 10%，是翻倍。因为安阳的孩子，从来不怕天高地远。
